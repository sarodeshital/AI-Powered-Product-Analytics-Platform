# AI-Powered Product Review Analytics & Business Intelligence System
## Project Overview
This project is an AI-driven customer review analytics and business intelligence platform designed to analyze large-scale product reviews, identify customer sentiment, detect product-related issues, and generate actionable business insights.

The system processes unstructured customer feedback using NLP techniques, stores analytical data in SQL, and visualizes insights using interactive Power BI dashboards.

## Key Features
Sentiment Analysis
Product Issue Detection
Severity Index Calculation
Product AI Scoring
Product Ranking System
Risk Category Identification
SQL-Based Analytical Storage
Interactive Power BI Dashboards
Business Intelligence Reporting
